# BalmJS template - vue-expert

> A Vue.js project with router

## Installation

```sh
npm install -g balm-cli
balm init vue-expert my-project

cd my-project
yarn
# or
npm i
```

> (Recommended) [BalmUI](https://material.balmjs.com/) for your project

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

> thx [BalmJS](https://balmjs.com/)
